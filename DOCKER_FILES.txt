QUICK START:
============

1. Install Docker and Docker Compose on your system

2. Navigate to project:
   cd /home/am/Dokumente/develop/finclusion

3. Start with docker-compose:
   docker-compose up -d

4. Visit: http://localhost:8080
   Login: admin / admin

5. View logs:
   docker-compose logs -f

6. Stop:
   docker-compose down

BUILD OPTIONS:
===============

Option 1 - Docker Compose (Recommended):
  docker-compose up -d

Option 2 - Manual Build & Run:
  docker build -t fixhub:latest .
  docker run -d -p 8080:8080 fixhub:latest

Option 3 - Helper Script:
  ./docker-helper.sh build
  ./docker-helper.sh compose-up

ENVIRONMENT VARIABLES:
======================

Required (optional with defaults):
  APP_JWT_SECRET              (JWT signing key)
  APP_JWT_EXPIRATION         (24 hours default)

Optional:
  JAVA_OPTS                  (JVM settings, -Xmx512m default)
  FIXHUB_CONFIG_PATH         (config file path)
  FIXHUB_CONNECTIONS_FILE    (connections file)
  LOG_LEVEL                  (INFO default)

VOLUMES:
========

/app/store      - FIX message store and persistent data

PORTS:
